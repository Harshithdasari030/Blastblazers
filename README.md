College Prediction Using Rank, Location & Fees
📌 Project Description

This project is an AI-powered college recommendation system that helps students predict and find suitable colleges based on their entrance rank, preferred location, budget/fees, and other academic preferences. The system reduces admission confusion by providing personalized college suggestions.

🚀 Features

Rank-based college prediction

Location preference filtering

Fee/budget-based recommendation

Personalized college ranking

User-friendly web interface

Fast AI-driven suggestions

🧠 Technologies Used

Python

Streamlit (Frontend UI)

Pandas & NumPy

Machine Learning Algorithms

Scikit-learn

📂 Dataset
The dataset includes:
College name
Entrance rank cutoff
Course fees

Location

Branch/stream availability
